projects

Here is a listing of some projects that I’ve worked on over the years.

  • SolarViewer — A viewer for the Sun’s coronal magnetic field, built with WebGL using the Three.js library during a job I had at the Lockheed Martin Advanced Technology Center in Palo Alto in my senior year of high school.

  • CodaLab — I worked on CodaLab for 3 months with Professor Percy Liang and a group of graduate students. In that time, I implemented a feature that improved the scalability of the backend by using consistent hashing to partition data across a set of disks in a fashion that allowed dynamically adding or removing disks from the storage set with minimal time spent rebalancing. Full set of contributions to the backend can be seen here. I also converted the system to use modern JS ecosystem of tools, including Bower and npm scripts for automation. PRs are here and here.

  • Consist — A consistent hashing library built in Rust. Available on crates.io.

  • Changelog Nightly Proxy — I host a proxy that redirects to the latest edition of the ChangeLog proxy. The code is really simple. Here’s a live version. EDIT: NO LONGER MAINTAINED.

  • disGo — A simple distributed process runner written in Go. It was originally created as a way to run and automatically restart a set of rendering processes for a final image in CS 148, but easily generalizes to running any type of distributed task with restarts.
